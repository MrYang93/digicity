首先，搭建一个基本的 Webpack-React 项目。


### 创建 Github 项目

```
mkdir redux-hello
```

然后 github.com 上也添加一个同名仓库。


### 要求

可以用

```
npm start
```

来启动项目。


可以加载 main.css 文件。

可以运行一个基本的 React 组件。
