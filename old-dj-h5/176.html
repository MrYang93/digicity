transition 过度效果

UI 设计，也就是用户界面的设计，不仅仅是个静态的美，因为界面是动的。所以好得用户体验很大程度来自于动态效果。相关的 CSS/JS 技巧不少，今天介绍最重要的一个，就是 transition 。

案例展示

先来看看过渡效果能做出什么效果。

  • 第一个，教学案例的导航栏和侧边栏（ http://meteor.haoduoshipin.com ）。

  • 第二个，这个按钮效果中 也用到了 transiton 。

  • 第三个，不依赖于 JS 的菜单 。

基本语法

代码要写成这样

    transition: &#x8FC7;&#x6E21;&#x5C5E;&#x6027; &#x6301;&#x7EED;&#x65F6;&#x95F4; &#x6548;&#x679C;&#x51FD;&#x6570; &#x8FC7;&#x6E21;&#x5EF6;&#x65F6;;
}
