添加样式

如何让一个一级标题变成绿色：

第一种方式：行内样式

<h1 style="color: green"> hello </h1>

第二种方式：页内方式

<style>
 h1 {
   color: #bada33;
   font-size: 70px;
 }
</style>

第三种方式：独立的 css 文件

<link rel="stylesheet" href="main.css">

然后创建 main.css 来书写样式规则就好了。
