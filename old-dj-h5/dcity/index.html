第一天

和大家见面，开学典礼，互相介绍，了解一下大家的基础。然后跟大家一起安装 sublime 编辑器和 Chrome 浏览器。书写了一个 html 页面，逐步通过行内样式，页内样式引出了 css 文件。然后演示如何使用 Chrome 的开发者工具来临摹人家的页面。 进入

第二天

开始页面大块布局。首先介绍 div 和块元素，然后介绍了 span 和行内元素。接下来引入了页面中通常都会有的 header main footer 这三部分，通过三个 div 按照“一般流程”来摆放来实现了这三个部分。下面介绍如何把 main 这一部分分为左右两栏。使用 float 使得元素跳出一般流程。进入

第三天

讲解图片使用，用 text-align 将图片居中，用 margin: 0 auto 让块元素居中。介绍了“盒子模型”的 margin padding border 的概念。css 的复合选择符，介绍了逗号分隔符和空格分隔符各自的语法意义。让大家利用 gitcafe.com 的服务，拥有了自己的个人网站。进入

第四天

开始动手制作小元器件，今天制作的是一个个人 profile 卡片，在实际例子中引入 border-radius，box-shadow，clearfix 以及 github 客户端使用的知识。进入

第五天

继续动手制作元器件，今天制作的是页首导航栏。下午介绍了在 sublime 中安装 emmet 插件，以及如何使用 github 客户端操作 gitcafe 上的项目。进入

第六天

花了很长时间让大家学习使用 github 客户端。内容很繁杂，涉及命令行使用，相信大家的学习体验不是特别愉悦。但是当最终下午大家可以自如的上传自己的内容到 gitcafe.com 的时候，确实是很帅的。今天的知识点主要是字体图标。进入

第七天

开始切一张 psd 图，主角是如何用 photoshop 量尺寸，抠图片。CSS 方面主要是 background 的使用。 进入

第八天

继续切图。让大家在综合场景中去熟悉盒子模型。Peter 做的工作，今天主要不是直接灌输给大家，而是让大家自己动手，然后 Peter 个别跟每位同学交流，讨论代码可以改进的地方。主要涉及到的知识点是 float 的使用，如何加载自定制字体，如何进行大块布局规划。 进入

第九天

早上给大家介绍了开源软件的概念，演示 input 的样式设置。然后介绍了 postion:relative 的用法。下午，介绍了代码复用的基本思想，并演示了 Bootstrap 的基本使用。 进入

第十天

最后一天了，上午大家考试也没上课。下午给大家介绍了程序员每天都会用到的各种资源，例如 google wikipedia 等等。进入
